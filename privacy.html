Política de Privacidad

Fecha de entrada en vigencia: 22 de junio de 2025

Esta aplicación ha sido desarrollada con el objetivo de permitir a los usuarios expresar su apoyo a un candidato presidencial mediante un sistema de votación única por dispositivo.

Información que recopilamos

Para garantizar la integridad del proceso de votación, la aplicación genera una clave única basada en los siguientes datos del dispositivo:

  • Identificador del dispositivo
  • Modelo del dispositivo
  • Marca del dispositivo
  • Sistema operativo

No recopilamos nombres, correos electrónicos, números de teléfono ni ninguna otra información personal del usuario.

Uso de la información

La clave generada a partir de los datos del dispositivo se utiliza únicamente para identificar de forma anónima un voto por dispositivo. Esta clave no se puede usar para identificar al usuario individualmente.

Seguridad

Todos los datos se manejan de forma segura y cifrada. Además, la aplicación verifica que el dispositivo:

  • No esté rooteado
  • No sea un emulador
  • No esté en modo desarrollador

Estas medidas buscan prevenir el uso fraudulento de la aplicación.

Transparencia y anonimato

La votación es completamente anónima. La aplicación no almacena ni asocia la identidad del usuario con el voto emitido.
