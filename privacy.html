Política de Privacidad

Fecha de entrada en vigencia: 22 de junio de 2025

Esta aplicación ha sido desarrollada para permitir que los usuarios expresen su apoyo a un candidato presidencial mediante una votación única por dispositivo. Valoramos tu privacidad y nos comprometemos a proteger la información que recopilamos.

Datos que recopilamos

La aplicación genera una firma única del dispositivo utilizando únicamente información técnica, con el fin de asegurar que cada usuario solo pueda emitir un voto por dispositivo.

Los datos utilizados para generar esta clave son:

  • Identificador único del dispositivo (como Android ID o Identifier for Vendor en iOS)
  • Modelo del dispositivo
  • Marca del dispositivo
  • Versión del sistema operativo

No recopilamos ni almacenamos datos personales como nombre, dirección de correo electrónico, número de teléfono, ubicación o contenido del dispositivo.

Uso de la información

La información del dispositivo se utiliza exclusivamente para generar una clave única y anónima. Esta clave garantiza que solo se emita un voto por dispositivo. No se almacena la información original, solo el valor encriptado (hash).

Seguridad

Todos los datos se manejan de manera segura y se cifran localmente antes de ser procesados. Además, la aplicación implementa medidas de seguridad para garantizar la integridad del sistema de votación:

  • Detecta si el dispositivo está rooteado
  • Detecta si se está ejecutando en un emulador
  • Verifica si el modo desarrollador está activado

Transparencia y anonimato

La votación es completamente anónima. No hay forma de vincular la clave generada con la identidad del usuario. Solo se utiliza para validar que el dispositivo ha votado.

Cumplimiento con Google Play

Esta aplicación declara el uso de identificadores del dispositivo, tal como lo exige la política de Datos del Usuario de Google Play. No compartimos esta información con terceros.
